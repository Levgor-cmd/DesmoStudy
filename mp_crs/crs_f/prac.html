Практика

Рад приветствовать вас в разделе с практикой. Здесь представлены задания по 3 уровням сложности:

1) начинающий (суммарно рассчитаны на минут 5-7, т.к. крайне поверхностно охватывают материал);

2) продвинутый (задачки уже посложнее);

3) эксперт (полноценные проекты, с которыми уже придется конкретно повозиться);

Думаю, чтобы закрепить знания по курсу этого вполне хватит. Удачи в решении!

Начинающий (1 - 12):


1) Построить график функции f(x) = sin(x), изменив стиль линии на штрих
2) Задать уравнение окружности с центром в точке (0,3) и радиусом 4
3) Ограничить 3 неравенства (y = 3x, y = -3(x+1), y = 7) так, чтобы получился треугольник
4) Построить параболлу ax2 + bx + c, где a = -1, b = 2 и c = 5
ОБРАТИТЕ ВНИМАНИЕ: ползунки должны быть добавлены в график
5) Используя таблицу, построить квадрат со стороной 5, левый нижний угол которого располагается в точке с координатами (3, 2)
6) Используя список, построить квадрат из задания 5
7) Воспроизвести ноту Ля первой октавы
8) Вычислить число сочетний C25
9) Найти расстояние между точками (4,3) и (-2, 9)
10) Вычислить сумму с 5 по 12 члены арифметической прогрессии an+1= an + 35 при условии что a3 = 5
11) Построить график функции сигнум для y = 4x - 5
12) Найти остаток от деления числа 45200 на 36

Продвинутый (13 - 20):


1) Создать точку, которая будет двигаться по кривой, заданной графиком функции y = cosh(x) (границы ползунка, отвечающего за движение точки [-5; 5])
2) Построить кусочно-заданную функцию, задав ее одним выражением:

3) Анимировать движение точки по окружности с радиусом 1 и центром в точке (0,0)
4) Создать список lst = [1,2,3 … +∞], длина которого будет увеличиваться на 1 каждую секунду (изначально lst = [1])
5) Построить семейство тангенсоид y = -3 tg x, y = -2 tg x … y = 3 tg x и раскрасьте каждую соответсвующим цветом: hsv(120, 1, 1), hsv(125, 1, 1) … hsv(150, 1, 1)
6) Реализовать гамму тональности c-moll, подобную примеру
7) Создать анимацию изменения коэффициентов параболы ax^2 + bx + c с испольованием интерактиных кнопок
8) Используя уравнение касательной и изображение лыжника, анимировать спуск с горки, заданной графиком функции f(x) = ½ (sin x - x)

Эксперт (21 - 25):


1) Реализовать рабочую модель часов (циферблат можете взять отсюда)
2) Анимировать движение машинки с вращением колес, используя изображения: колеса, кузов
3) Создать модель солнечной системы с анимацией вращения планет (обратите внимание на интервал времени, который требуется каждой планете для оборота вокруг солнца)
4) Реализовать рабочую модель часов с подсчетом времени в формате чч:мм:сс
5) Анимировать вращение додекаэдра по одной из 15 осей
